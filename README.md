# Piltaland HomePage

🏛️ **ピルタランド共和国の公式ホームページ**

架空国家Piltalandの静的ホームページです。GitHub Pagesでホストされています。

## 概要

このサイトはPiltalandという架空の国家に関する情報を提供する静的Webサイトです。

## 特徴

- レスポンシブデザイン（スマートフォン、タブレット、デスクトップに対応）
- スムーズスクロールナビゲーション
- モダンなUIデザイン
- アニメーション効果

## ファイル構成

```
.
├── index.html    # メインHTMLファイル
├── styles.css    # スタイルシート
├── script.js     # JavaScriptファイル
└── README.md     # このファイル
```

## 使用技術

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ローカルでの表示方法

1. このリポジトリをクローン
2. `index.html`をブラウザで開く

または、簡易的なHTTPサーバーを起動：

```bash
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

## GitHub Pagesでの公開

このサイトはGitHub Pagesで公開されています。リポジトリの設定でGitHub Pagesを有効にすることで、自動的にデプロイされます。

## コンテンツ
